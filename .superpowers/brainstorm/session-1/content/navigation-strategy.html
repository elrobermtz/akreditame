Estrategia de Navegación y Verticales

¿Cómo quieres presentar las diferentes divisiones (Energía, Infraestructura, etc.)?

A

Enfoque Unificado

Una sola página donde todas las verticales se mencionan como "Soluciones". Todo lleva al mismo formulario de contacto principal.

Hero -> Servicios (A, B, C) -> Formulario
B

Hub de Verticales

Secciones dedicadas con mayor profundidad para cada vertical (Energía, Infra, etc.) con beneficios específicos para cada tipo de activo.

Hero -> [Energía] [Infra] [Real Estate] -> Contacto
C

Explorador Interactivo

Un selector tipo "¿Qué buscas?" (Invertir vs. Desarrollar Proyecto) que adapta el contenido dinámicamente según el perfil del usuario.

Hero -> [Inversionista] | [Desarrollador] -> Contenido a medida
