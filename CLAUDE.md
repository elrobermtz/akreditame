# Superpowers Configuration

Este proyecto utiliza el framework **Superpowers**.

## Ubicación de Habilidades
Las habilidades (skills) se encuentran en `.agent/skills/`.

## Instrucciones para Agentes
Cualquier agente que trabaje en este repositorio (Claude Code, Gemini, Antigravity) DEBE:
1. Leer y seguir las instrucciones de `using-superpowers` en `.agent/skills/using-superpowers/SKILL.md`.
2. Usar las herramientas correspondientes para activar las habilidades antes de proceder.

### Paralelismo (Claude Code)
Utiliza la herramienta `Task()` para despachar subtareas independientes.
